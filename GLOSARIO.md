# Glosario de Términos - Fusion 360

## A

**Análisis de Tensiones (Stress Analysis)**
Simulación que calcula las tensiones, deformaciones y factores de seguridad en un diseño bajo cargas específicas.

**Appearance (Apariencia)**
Propiedades visuales de un objeto (color, textura, brillo) que no afectan la geometría pero mejoran la presentación.

**Assembly (Ensamblaje)**
Colección de componentes posicionados y relacionados entre sí para formar un producto completo.

**Axis (Eje)**
Línea de referencia utilizada para operaciones como revolución, patrones circulares, o para definir sistemas de coordenadas.

## B

**Body (Cuerpo)**
Geometría sólida o de superficie. Un componente puede contener múltiples bodies.

**Boolean Operation (Operación Booleana)**
Operaciones de unión (join), resta (cut) o intersección (intersect) entre sólidos.

**Boundary Fill**
Crear un sólido o superficie cerrando un volumen definido por superficies existentes.

**Browser**
Panel lateral que muestra la jerarquía del diseño: componentes, bodies, sketches, construcciones, etc.

## C

**CAD (Computer-Aided Design)**
Diseño asistido por computadora. Uso de software para crear, modificar y analizar diseños.

**CAM (Computer-Aided Manufacturing)**
Manufactura asistida por computadora. Generación de trayectorias de herramientas para mecanizado CNC.

**Canvas**
Área de trabajo principal donde se visualiza y manipula el modelo 3D.

**Chamfer (Chaflán)**
Operación que crea un bisel o corte angular en una arista.

**Component (Componente)**
Unidad de diseño que puede contener bodies, sketches y otros elementos. Fundamental en ensamblajes.

**Constraint (Restricción)**
Relación geométrica que define cómo los elementos del sketch se relacionan entre sí (paralelo, perpendicular, tangente, etc.).

**Construction (Construcción)**
Geometría auxiliar (líneas, planos, puntos) utilizada como referencia pero que no forma parte del modelo final.

## D

**Dimension (Dimensión)**
Valor numérico que define el tamaño o posición de elementos en el diseño.

**Draft (Ángulo de Desmoldeo)**
Ángulo agregado a caras para facilitar la extracción de piezas de moldes.

**DXF/DWG**
Formatos de archivo de AutoCAD, comúnmente utilizados para intercambio de dibujos 2D.

## E

**Extrude (Extrusión)**
Operación que convierte un perfil 2D en un sólido 3D al darle profundidad.

## F

**Feature (Característica)**
Operación de modelado que modifica la geometría (extrusión, redondeo, patrón, etc.).

**Fillet (Redondeo)**
Operación que suaviza aristas creando una transición redondeada.

**Fusion 360 (.f3d)**
Formato de archivo nativo de Fusion 360 que preserva todo el historial y parámetros del diseño.

## G

**G0, G1, G2 (Continuidad)**
Niveles de continuidad entre superficies:
- **G0**: Continuidad de posición (superficies conectadas)
- **G1**: Continuidad tangente (transición suave)
- **G2**: Continuidad de curvatura (transición muy suave)

**GD&T (Geometric Dimensioning and Tolerancing)**
Sistema de símbolos para especificar tolerancias geométricas en planos técnicos.

**Grounded (Anclado)**
Componente fijo en su posición en un ensamblaje, que no se puede mover.

## H

**HDRI (High Dynamic Range Imaging)**
Imágenes especiales utilizadas para iluminación realista basada en imágenes.

**History (Historial)**
Ver Timeline.

## I

**IGES (Initial Graphics Exchange Specification)**
Formato de archivo neutro para intercambio de datos CAD entre diferentes sistemas.

**Instance (Instancia)**
Copia de un componente que comparte la misma geometría que el original. Cambios al original afectan todas las instancias.

## J

**Joint (Unión)**
Conexión entre componentes en un ensamblaje que define cómo pueden moverse relativamente.

## L

**Loft**
Operación que crea una superficie o sólido interpolando entre múltiples perfiles.

## M

**Mesh (Malla)**
Geometría compuesta por vértices, aristas y caras triangulares o poligonales. Menos flexible que geometría sólida.

**Mirror (Simetría)**
Crear una copia reflejada de geometría respecto a un plano.

**Modeling Workspace (Espacio de Modelado)**
Ver Workspace.

## O

**Offset (Desfase)**
Crear una copia paralela de una curva, superficie o plano a una distancia específica.

**Orbit (Órbita)**
Rotar la vista 3D alrededor del modelo.

**Origin (Origen)**
Punto de referencia (0,0,0) del sistema de coordenadas.

## P

**Pan (Desplazar)**
Mover la vista sin cambiar el ángulo de visualización.

**Parameter (Parámetro)**
Variable numérica que controla dimensiones en el modelo. Permite diseño paramétrico.

**Parametric Modeling (Modelado Paramétrico)**
Método de modelado donde el historial de operaciones se mantiene y puede editarse.

**Pattern (Patrón)**
Repetición de features en arreglos rectangulares, circulares o a lo largo de trayectorias.

**Plane (Plano)**
Superficie plana infinita utilizada como referencia para sketches y operaciones.

**Press Pull**
Herramienta de modelado directo para empujar/tirar caras sin usar el timeline.

## R

**Ray Tracing (Trazado de Rayos)**
Técnica de renderizado que simula el comportamiento físico de la luz para imágenes fotorrealistas.

**Render (Renderizado)**
Proceso de generar una imagen fotorrealista del modelo con materiales e iluminación.

**Revolve (Revolución)**
Operación que crea un sólido rotando un perfil alrededor de un eje.

**Rigid Group**
Conjunto de componentes que se mueven juntos como si fueran uno solo.

## S

**Section View (Vista de Sección)**
Vista que muestra el interior de un modelo como si hubiera sido cortado.

**Shell (Vaciado)**
Operación que crea paredes de espesor uniforme removiendo material del interior de un sólido.

**Sketch**
Dibujo 2D que sirve como base para operaciones 3D. Puede contener líneas, arcos, círculos, etc.

**Spline**
Curva suave que pasa por una serie de puntos de control.

**STEP (Standard for the Exchange of Product Data)**
Formato de archivo estándar ISO para intercambio de datos CAD 3D.

**STL (STereoLithography)**
Formato de archivo de malla utilizado principalmente para impresión 3D.

**Surface Modeling (Modelado de Superficies)**
Técnica de modelado que crea y manipula superficies sin grosor, útil para formas complejas.

**Sweep (Barrido)**
Operación que crea un sólido o superficie barriendo un perfil a lo largo de una trayectoria.

## T

**Tangent (Tangente)**
Restricción que hace que dos curvas se toquen suavemente sin esquina.

**Thread (Rosca)**
Feature que crea roscas en agujeros o ejes cilíndricos.

**Timeline**
Línea de tiempo en la parte inferior de la interfaz que muestra el historial de operaciones del diseño.

**T-Spline**
Tipo especial de superficie subdividida útil para formas orgánicas.

## U

**Undercut (Socavado)**
Característica geométrica que previene la extracción de una pieza de un molde sin daño.

**User Parameters (Parámetros de Usuario)**
Parámetros definidos por el usuario para controlar el diseño de manera flexible.

## V

**ViewCube**
Control de navegación en esquina superior derecha que permite orientar rápidamente la vista.

## W

**Workspace (Espacio de Trabajo)**
Entorno de trabajo especializado en Fusion 360:
- **Design**: Modelado 3D
- **Render**: Renderizado
- **Animation**: Animación
- **Simulation**: Análisis
- **Manufacture**: CAM
- **Drawing**: Planos 2D

## X

**XY/YZ/XZ Plane**
Planos de referencia principales basados en el sistema de coordenadas cartesianas.

## Z

**Zebra Stripes (Rayas de Cebra)**
Análisis visual que muestra la suavidad de superficies mediante líneas de reflexión.

**Zoom**
Acercar o alejar la vista del modelo.
