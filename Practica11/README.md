# Práctica 11: Renderizado y Presentación

## Objetivo
Aprender a crear imágenes fotorrealistas y presentaciones profesionales de diseños.

## Descripción
Técnicas de renderizado y presentación para comunicar diseños de manera efectiva.

## Contenido de la Práctica

### 1. Espacio de Trabajo Render
- Configuración de cámara
- Perspectiva y proyecciones
- Composición de la escena

### 2. Materiales y Apariencias
- Biblioteca de materiales
- Crear materiales personalizados
- Propiedades físicas (reflexión, refracción, rugosidad)
- Mapas de textura

### 3. Iluminación
- Iluminación ambiental
- Luces direccionales
- Luces puntuales y de área
- HDRI y Image-Based Lighting

### 4. Configuración de Render
- Calidad de renderizado
- Resolución de salida
- Ray tracing
- Render en la nube

### 5. Presentación
- Crear explosiones de ensamblajes
- Animaciones simples
- Hojas de presentación

## Ejercicios
1. Renderizar una pieza con diferentes materiales
2. Crear una escena iluminada profesionalmente
3. Generar una presentación de producto

## Archivos
- `escena_render.f3d`: Escena de ejemplo
- `materiales_personalizados/`: Materiales custom
- `hdri_ejemplos/`: Imágenes HDRI para iluminación

## Tiempo Estimado
75 minutos
