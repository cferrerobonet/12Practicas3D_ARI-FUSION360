# Práctica 08: Modelado de Superficies

## Objetivo
Aprender técnicas de modelado de superficies para crear formas complejas y orgánicas.

## Descripción
Esta práctica introduce el modelado de superficies (Surface Modeling) para geometrías avanzadas.

## Contenido de la Práctica

### 1. Creación de Superficies
- Extrude de superficies
- Revolve de superficies
- Loft entre curvas
- Sweep de perfiles

### 2. Manipulación de Superficies
- Extend (extender superficies)
- Trim (recortar superficies)
- Stitch (coser superficies)
- Offset de superficies

### 3. Análisis de Superficies
- Continuidad G0, G1, G2
- Curvatura y zebra stripes
- Draft analysis

### 4. Conversión a Sólidos
- Thicken (dar espesor)
- Boundary Fill (rellenar)
- Cerrar volúmenes

## Ejercicios
1. Crear una botella de forma orgánica
2. Diseñar una carcasa de producto
3. Modelar una carrocería simple
