# Práctica 01: Introducción a la Interfaz de Fusion 360

## Objetivo
Familiarizarse con la interfaz de usuario de Autodesk Fusion 360 y sus herramientas básicas de navegación.

## Descripción
En esta práctica inicial, aprenderás a:
- Navegar por la interfaz de Fusion 360
- Conocer los diferentes espacios de trabajo
- Utilizar las herramientas de visualización (zoom, pan, orbit)
- Configurar las preferencias básicas del programa

## Contenido de la Práctica

### 1. Espacios de Trabajo
- **Design**: Para modelado 3D y diseño de piezas
- **Render**: Para crear imágenes fotorrealistas
- **Animation**: Para crear animaciones de ensamblajes
- **Simulation**: Para análisis de tensiones
- **Manufacture**: Para generación de trayectorias de mecanizado
- **Drawing**: Para crear planos técnicos 2D

### 2. Navegación 3D
- **Orbit**: Clic medio del ratón o Shift + Clic medio
- **Pan**: Clic medio + Shift
- **Zoom**: Rueda del ratón

### 3. ViewCube
Herramienta para orientar rápidamente la vista hacia caras, aristas o esquinas estándar.
