# Práctica 07: Diseño Paramétrico y Parámetros

## Objetivo
Dominar el uso de parámetros y diseño paramétrico para crear modelos adaptativos.

## Descripción
Aprenderás a crear diseños que se adaptan automáticamente al cambiar parámetros clave.

## Contenido de la Práctica

### 1. Parámetros de Usuario
- Crear parámetros personalizados
- Tipos de parámetros (longitud, ángulo, cantidad, etc.)
- Nomenclatura y organización

### 2. Ecuaciones y Expresiones
- Operaciones matemáticas básicas
- Funciones trigonométricas
- Referencias entre parámetros

### 3. Tablas de Parámetros
- Crear familias de piezas
- Configuraciones múltiples
- Exportar e importar tablas

### 4. Diseño por Intención
- Capturar la intención de diseño
- Restricciones inteligentes
- Robustez del modelo

## Ejercicios
1. Crear una familia de tornillos paramétricos
2. Diseñar una pieza que escala proporcionalmente
3. Implementar un diseño con configuraciones múltiples

## Archivos
- `modelo_parametrico.f3d`: Ejemplo de modelo paramétrico
- `tabla_configuraciones.csv`: Tabla de parámetros
- `ejercicio_parametrico.pdf`: Ejercicio guiado

## Tiempo Estimado
75 minutos
