# Práctica 12: Proyecto Integrador Final

## Objetivo
Aplicar todos los conocimientos adquiridos en un proyecto completo de diseño.

## Descripción
Proyecto final que integra todas las técnicas aprendidas en las prácticas anteriores.

## Contenido de la Práctica

### Proyecto: Diseño Completo de un Producto

Elige una de las siguientes opciones o propón tu propio proyecto:

1. **Carcasa de Dispositivo Electrónico**
   - Diseño ergonómico
   - Ensamblaje de tapa y base
   - Soportes internos
   - Ventilación

2. **Herramienta Mecánica**
   - Mecanismo funcional
   - Piezas múltiples
   - Análisis de movimiento
   - Planos de manufactura

3. **Mueble o Accesorio**
   - Diseño estético
   - Ensamblaje de componentes
   - Visualización fotorrealista
   - Planos de fabricación

### Requisitos del Proyecto

1. **Modelado 3D Completo**
   - Mínimo 5 componentes
   - Uso de técnicas variadas
   - Diseño paramétrico

2. **Ensamblaje**
   - Joints funcionales
   - Análisis de interferencias
   - Explosión de ensamblaje

3. **Documentación Técnica**
   - Planos 2D de cada componente
   - Plano de conjunto
   - Lista de materiales (BOM)

4. **Presentación**
   - Renders fotorrealistas (mínimo 3)
   - Diferentes vistas y ángulos
   - Presentación profesional

### Entregables

- Archivo .f3d del modelo completo
- Planos PDF de cada componente
- Renders de presentación
- Documento de memoria del proyecto

## Criterios de Evaluación

- **Complejidad técnica**: 25%
- **Calidad del modelado**: 25%
- **Documentación**: 25%
- **Presentación visual**: 25%

## Tiempo Estimado
4-6 horas (puede realizarse en varias sesiones)

## Recursos de Apoyo
- Todas las prácticas anteriores
- Foros de Autodesk
- Galería de proyectos de inspiración
