# Práctica 09: Modelado de Chapa Metálica

## Objetivo
Aprender técnicas específicas para diseño de piezas de chapa metálica.

## Descripción
Técnicas especializadas para diseño y manufactura de componentes de chapa.

## Contenido de la Práctica

### 1. Herramientas de Sheet Metal
- Flange (pestaña)
- Bend (doblado)
- Hem (dobladillo)
- Contour Flange
- Contour Roll

### 2. Operaciones de Corte
- Rectangular Pattern Cut
- Circular Pattern Cut
- Corner Relief (alivio de esquina)
- Bend Relief (alivio de doblado)

### 3. Reglas de Sheet Metal
- Grosor de material
- Radio de doblado mínimo
- Factor K
- Relief styles

### 4. Flat Pattern (Desarrollo)
- Obtener el desarrollo plano
- Exportar para corte CNC
- Cálculo de longitud desarrollada

## Ejercicios
1. Diseñar una caja de chapa
2. Crear un soporte en L
3. Modelar una carcasa compleja

## Archivos
- `chapa_ejemplo.f3d`: Ejemplos de chapa
- `desarrollo_plano.dxf`: Ejemplo de desarrollo
- `ejercicio_chapa.pdf`: Ejercicio práctico

## Tiempo Estimado
75 minutos
