# Práctica 10: Creación de Planos Técnicos 2D

## Objetivo
Aprender a crear planos técnicos 2D profesionales a partir de modelos 3D.

## Descripción
Esta práctica enseña a generar documentación técnica completa para manufactura.

## Contenido de la Práctica

### 1. Configuración del Plano
- Tamaños de papel estándar (A4, A3, A2, A1, A0)
- Cajetín y cuadro de rotulación
- Escalas de dibujo

### 2. Vistas de Dibujo
- Vista Base
- Vistas Proyectadas
- Vistas Auxiliares
- Vistas de Sección
- Vistas de Detalle

### 3. Acotación
- Acotación lineal
- Acotación angular
- Acotación de radios y diámetros
- Tolerancias dimensionales
- Tolerancias geométricas (GD&T)

### 4. Anotaciones
- Notas generales
- Referencias de superficie
- Símbolos de soldadura
- Lista de materiales (BOM)

## Ejercicios
1. Crear un plano de conjunto
2. Generar plano de despiece
3. Acotar completamente una pieza mecánica

## Archivos
- `plantilla_plano.dwg`: Plantilla de plano
- `ejemplo_plano.pdf`: Ejemplo de plano completo
- `ejercicio_plano.pdf`: Ejercicio a realizar

## Tiempo Estimado
90 minutos
