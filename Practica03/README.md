# Práctica 03: Modelado Básico - Extrusion y Revolve

## Objetivo
Dominar las operaciones básicas de modelado 3D: extrusión y revolución.

## Descripción
Aprenderás a convertir bocetos 2D en sólidos 3D mediante las operaciones más fundamentales de modelado.

## Contenido de la Práctica

### 1. Extrusión (Extrude)
- **Extrusión simple**: Dar profundidad a un sketch
- **Extrusión con ángulo**: Crear formas cónicas
- **Tipos de operación**:
  - Join (unir)
  - Cut (cortar)
  - Intersect (intersectar)
  - New Body (nuevo cuerpo)

### 2. Revolución (Revolve)
- Crear sólidos de revolución a partir de un perfil
- Definir el eje de revolución
- Ángulos completos (360°) y parciales

### 3. Operaciones Booleanas
- Unión de sólidos
- Sustracción de material
- Intersección de volúmenes

## Ejercicios
1. Modelar un cilindro mediante extrusión
2. Crear un vaso mediante revolución
3. Diseñar una pieza mecánica combinando extrusiones
