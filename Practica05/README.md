# Práctica 05: Modificación de Sólidos - Fillets y Chamfers

## Objetivo
Aprender a modificar y refinar modelos 3D mediante redondeos, chaflanes y shells.

## Descripción
Técnicas para agregar detalles y refinamientos a los modelos sólidos.

## Contenido de la Práctica

### 1. Fillet (Redondeo)
- Redondeos de radio constante
- Redondeos de radio variable
- Redondeos de cara completa
- Consideraciones de manufactura

### 2. Chamfer (Chaflán)
- Chaflanes de distancia igual
- Chaflanes de dos distancias
- Chaflanes por ángulo y distancia
- Aplicaciones en diseño mecánico

### 3. Shell (Vaciado)
- Crear paredes de espesor uniforme
- Remover caras para crear cavidades
- Aplicaciones en diseño de contenedores

### 4. Draft (Ángulo de Desmoldeo)
- Agregar ángulos de salida para moldeo
- Aplicación en piezas de inyección de plástico

## Ejercicios
1. Aplicar redondeos a una pieza mecánica
2. Crear una carcasa mediante shell
3. Diseñar una pieza con chaflanes funcionales

- `pieza_modificaciones.f3d`: Ejemplos de modificaciones
- `ejercicio_refinamiento.pdf`: Ejercicio práctico

60 minutos
