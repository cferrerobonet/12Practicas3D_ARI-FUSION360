# Práctica 04: Operaciones de Modelado Avanzado

## Objetivo
Aprender operaciones avanzadas de modelado como loft, sweep y patrones.

## Descripción
Esta práctica introduce técnicas más sofisticadas para crear geometrías complejas.

## Contenido de la Práctica

### 1. Loft
- Crear superficies entre múltiples perfiles
- Controlar la transición con guías
- Aplicaciones en diseño orgánico

### 2. Sweep
- Barrer un perfil a lo largo de una trayectoria
- Crear tubos y perfiles complejos
- Opciones de orientación del perfil

### 3. Patrones (Patterns)
- **Patrón Rectangular**: Repetir features en filas y columnas
- **Patrón Circular**: Repetir alrededor de un eje
- **Patrón en Trayectoria**: Seguir una curva

### 4. Mirror (Simetría)
- Reflejar features respecto a un plano
- Crear diseños simétricos eficientemente

## Ejercicios
1. Crear un resorte mediante sweep
2. Diseñar una hélice usando loft
3. Aplicar patrones circulares para crear engranajes
