# Práctica 02: Sketching y Restricciones

## Objetivo
Aprender a crear bocetos 2D (sketches) y aplicar restricciones geométricas y dimensionales.

## Descripción
Esta práctica se centra en el dominio de las herramientas de bocetado, fundamentales para el modelado 3D paramétrico.

## Contenido de la Práctica

### 1. Herramientas de Sketch
- **Línea**: Crear líneas rectas
- **Círculo**: Dibujar círculos por centro o por 3 puntos
- **Arco**: Crear arcos de diferentes tipos
- **Rectángulo**: Dibujar rectángulos y cuadrados
- **Polígono**: Crear polígonos regulares
- **Spline**: Dibujar curvas suaves

### 2. Restricciones Geométricas
- **Horizontal/Vertical**: Forzar líneas horizontales o verticales
- **Perpendicular**: Hacer líneas perpendiculares entre sí
- **Paralelo**: Mantener líneas paralelas
- **Tangente**: Crear tangencias entre líneas y curvas
- **Coincidente**: Hacer que puntos coincidan
- **Concéntrico**: Centrar círculos o arcos

### 3. Restricciones Dimensionales
- Dimensiones lineales
- Dimensiones angulares
- Dimensiones de radio y diámetro
- Parámetros y ecuaciones

## Ejercicios
1. Dibujar un perfil básico de una pieza mecánica
2. Aplicar restricciones para hacer el sketch totalmente definido
3. Modificar dimensiones y observar el comportamiento paramétrico
